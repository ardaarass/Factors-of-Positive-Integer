Program Hakkında:

Bu program aşağıda belirtilen kurallar dahilinde kullanıcıdan alınan pozitif bir tamsayının çarpanlarını ekranda görüntülemektedir.

Programın Çalışma Şartları:

a)Bölenlerinin bulunmasını istediğiniz sayı pozitif bir tamsayı olmalıdır.

Nasıl Çalıştırılır?:

1)Çarpanlarını bulmak istediğiniz sayıyı giriniz.

2)Pozitif bir tamsayı girmemeniz durumunda program sizi uyaracaktır.

3)Sayının çarpanları ekranınızda görüntülenir.
